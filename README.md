# TestMachine + Sandbox

## О проекте

TestMachine + Sandbox — прототип AI-системы для структурированного анализа рабочих сценариев и ролевого поведения в организации.

Проект был разработан как MVP для заказчика. Основная задача системы — собирать данные из диалогов, преобразовывать их в структурированные JSON-артефакты и передавать подготовленные сигналы в аналитический модуль для дальнейшей обработки.

Система построена по принципу разделения ответственности: один модуль отвечает за сбор и структурирование данных, второй — за аналитическую обработку и построение гипотез.

## Цель проекта

Цель проекта — создать воспроизводимый AI-пайплайн, который помогает переводить неструктурированные диалоги в проверяемые, трассируемые и пригодные для анализа данные.

Особое внимание уделялось:

* структурированному сбору данных;
* evidence-based подходу;
* трассируемости выводов;
* сохранению артефактов каждого прогона;
* проверке качества данных;
* подготовке системы к дальнейшему пилотированию.

## Архитектура

Проект состоит из двух основных контуров:

### TestMachine

Модуль отвечает за первичную обработку:

* проведение структурированных диалогов;
* извлечение наблюдаемых поведенческих признаков;
* формирование JSON-карточек;
* подготовку структурированных сигналов;
* сохранение артефактов прогона;
* проверку полноты и качества данных.

### Sandbox

Модуль отвечает за аналитическую обработку:

* получение подготовленных сигналов;
* построение аналитических гипотез;
* моделирование возможных сценариев;
* подготовку данных для последующей экспертной оценки.

## Упрощённый пайплайн

```text
Dialog collection
        ↓
Structured profile card
        ↓
Role-based behavior card
        ↓
Signals pack
        ↓
Sandbox analysis
        ↓
Expert review
```
