# TestMachine + Sandbox

## О проекте

TestMachine + Sandbox — индивидуальный AI-проект по разработке архитектуры MVP-системы для структурированного анализа рабочих сценариев и ролевого поведения в организации.

Проект направлен на создание воспроизводимого пайплайна, который переводит неструктурированные диалоги в структурированные JSON-артефакты и аналитические сигналы для дальнейшей экспертной интерпретации.

Ключевая идея проекта — разделить систему на два контура:

* **TestMachine** — сбор, структурирование и проверка данных;
* **Sandbox** — аналитическая обработка сигналов и построение гипотез.

## Цель проекта

Цель проекта — спроектировать AI-пайплайн, который позволяет:

* проводить структурированные диалоги;
* извлекать наблюдаемые поведенческие признаки;
* формировать JSON-карточки;
* готовить структурированные сигналы;
* сохранять артефакты каждого прогона;
* обеспечивать трассируемость выводов;
* проверять качество и полноту данных.

## Архитектура

Проект состоит из двух основных модулей.

### TestMachine

TestMachine отвечает за первичную обработку данных:

* проведение диалогов;
* сбор наблюдаемых фактов;
* формирование структурированных карточек;
* подготовку сигналов;
* сохранение результатов прогона;
* проверку качества данных.

### Sandbox

Sandbox отвечает за аналитический слой:

* получение подготовленных сигналов;
* построение аналитических гипотез;
* моделирование возможных сценариев;
* подготовку данных для экспертной оценки.

## Упрощённый пайплайн

```text
Dialog collection
        ↓
Structured profile card
        ↓
Role-based behavior card
        ↓
Signals pack
        ↓
Sandbox analysis
        ↓
Expert review
```
